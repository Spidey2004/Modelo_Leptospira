# A Machine Learning Framework for Serogroup Classification of pathogenic species of Leptospira Based on rfb Locus Profiles

This repository gathers data used to generate machine learning models that enable the serological classification of pathogenic *Leptospira* species based solely on genomic data. 

The work is currently under review, but can be tested by those who have an assembled genome in FASTA format for this bacterium.

## Testing the framework
- The main workflow is implemented in a Colab notebook (`hierarchical_classifier_colab.ipynb`).
    - The notebook can be accessed in the following link: https://colab.research.google.com/drive/1PPU1BBlZMfFZnpr1beYSUjlNvePoTSSc?usp=sharing
- Users can open the notebook in Google Colab and execute the cells step by step.
- No local installation is required; everything runs in the cloud.

## Requirements
- A Google account to access Google Colab.

## Usage
1. Open the notebook in Google Colab.
2. Run the cells in order to reproduce the analysis and results.
